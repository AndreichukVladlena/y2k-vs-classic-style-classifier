# Fashion Style Classification (Classic vs Y2K)

Проект посвящён классификации изображений модных образов на два стиля: **classic** и **y2k** с использованием сверточной нейронной сети и transfer learning.

## Используемые подходы
- **Модель**: ResNet18 (предобученная на ImageNet)
- **Фреймворк**: PyTorch
- **Loss**: CrossEntropyLoss
- **Метрики**: Accuracy, F1-score (macro)
- **Оптимизация**: Adam
- **Валидация**: k-fold cross-validation (Stratified)

## Работа с данными
- Исходный датасет расширен с помощью **аугментаций** (повороты, кропы, отражения и др.)
- Аугментированные версии объединялись в один общий датасет
- k-fold кросс-валидация позволила избежать зависимости от одного разбиения train/val и стабилизировать метрики

## Результаты
- После добавления k-fold кросс-валидации метрики стали стабильными и в большинстве эпох достигали значений, близких к 1
- Модель хорошо работает на изображениях из интернета
- Основные ошибки связаны с пограничными случаями:
  - неконтрастные изображения часто относятся к *classic*
  - классические образы с яркими акцентами — к *y2k*
